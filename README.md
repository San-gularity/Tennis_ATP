# Tennis_ATP

## Prerequisites

This repository uses [Git LFS](https://git-lfs.github.com/) to handle large data files (CSV and image files). 

To clone this repository:

```bash
# Install Git LFS (if not already installed)
git lfs install

# Clone the repository
git clone https://github.com/San-gularity/Tennis_ATP.git
```

If you experience `remote-curl` buffer errors when pulling, ensure Git LFS is properly installed and initialized.
